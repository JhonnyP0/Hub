Project Description

This portfolio web application showcases my skills and projects as a developer. Built using Flask and MySQL, it provides a platform to demonstrate my proficiency in web development technologies.

Technologies Used

  • Flask
    A lightweight WSGI web application framework in Python.
  • Flask-WTF
    For form handling and validation.
  • Flask-Login
    For user session management.
  • Flask-SQLAlchemy
    For database interactions using SQLAlchemy ORM.
  • MySQL
    A popular relational database management system.
  • HTML/CSS
    For structuring and styling the web pages.
  • Gunicorn
    Python WSGI HTTP Server for UNIX.
  • Docker
    For containerizing the application.
